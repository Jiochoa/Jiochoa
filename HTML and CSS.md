## Some HTML to remember

### Lists
For **Unordered List** use `<ul>` with the **List Items** as `<li>`
#### input
```html
<ul>
	<li>element 1</li>
	<li>element 2</li>	
</ul>
```
<ul>
	<li>element 1</li>
	<li>element 2</li>	
</ul>
#### output
For **Ordered List** use `<ol>` with the **List Items** as `<li>`
```html
<ol>
	<li>element 1</li>
	<li>element 2</li>
</ol>
```
<ol>
	<li>element 1</li>
	<li>element 2</li>
</ol>
